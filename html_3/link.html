Penggunaan Link

Link adalah suatu cara dalam html untuk menghubungkan tulisan ke suatu alamat lain.
Beberapa contoh penggunaan link diantaranya:

  • Link ke github saya.
    Jika tulisan "github" tersebut di klik, maka anda akan diarahkan ke halaman github saya.
  • Link ke tentang saya.
    Jika tulisan "tentang" di klik, maka anda akan diarahkan ke halaman yang saya buat pada modul ke dua.
  • Link ke game saya.
    Jika tulisan "game" di klik, maka anda akan diarahkan ke game yang pernah saya buat pada UAS Mata Kuliah Multi Media semeseter 1 kemarin.
