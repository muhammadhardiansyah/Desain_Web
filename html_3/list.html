Macam-macam List

Terdapat dua macam list yang terdapat dalam html. Diantaranya :

  1. Unordered list

    2. Unordered list adalah list yang bentuknya berupa simbol.
    Contohnya:

    • Contoh Unordered list disc
    • Contoh Unordered list square
    • Contoh Unordered list circle
    • Contoh Unordered list none
  2. Ordered list

    3. Ordered list adalah list yang bentuknya berupa :

    • Angka, dengan menggunakan type = "1" .
      Contohnya:
      1. Contoh Ordered list 1
      2. Contoh Ordered list 2
      3. Contoh Ordered list 3
    • Huruf, dalam penggunaannya dapat dibagi lagi menjadi 2
      Yaitu:
      • Huruf besar, dengan menggunakan type = "A" .
        Contohnya:
        1. Contoh Ordered list A
        2. Contoh Ordered list B
        3. Conoth Ordered list C
      • Huruf kecil, dengan menggunakan type = "a" .
        Contohnya:
        1. Contoh Ordered list a
        2. Contoh Ordered list b
        3. Conoth Ordered list c
    • Angka Romawi , dalam penggunaanya juga dibagi menjadi 2.
      Yaitu:
      • Angka romawi besar dengan menggunakan type = "I"
        Contohnya:
        1. Contoh Ordered list I
        2. Contoh Ordered list II
        3. Contoh Ordered list III
      • Angka romawi kecil dengan menggunakan type = "i"
        Contohnya:
        1. Contoh Ordered list i
        2. COntoh Ordered list ii
        3. Contoh Ordered list iii
