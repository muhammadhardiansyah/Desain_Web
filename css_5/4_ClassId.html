Class & ID HTML

Dalam pemanggilan css terdapat dua selector, yaitu selector id dan selector class. Selector id ditandai dengan # dan selector class ditandai dengan titik (.).
Kedua selector ini memiliki perbedaan diantanya:

  • Class
    1. Dipakai untuk multiple element
    2. Dapat dipanggil sekaligus
    3. Menggunakan kode titik (.)
  • ID
    1. Dipakai untuk 1 element
    2. Muncul sekaligus
    3. Menggunakna kode #
